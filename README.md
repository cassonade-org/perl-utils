perl-utils 
---

Script *perl* pour ajouter un nouvel utilisateur sur cassonade.org. 
Rien d'extravagant, ça ajoute un user linux sur le serveur. 

installation 
---

`chmod +x install && ./install`

usage
---

`add_new_user`
